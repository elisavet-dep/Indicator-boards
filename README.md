# Indicator-boards
A set of display boards for embedded devices. The projects were completed as part of serial development.

## What's inside
- LED indicators (multiple revisions)
- Managed by interfaces and/or without them

## General Features
- Uniform size requirements
- Unified connectors
- Preparation for mass production

## My role
- Creation of an electrical circuit
- Tracing of the boards
- Preparation of the blank circuit boards
- Preparation of production files
- Creation of a 3D model
- Registration of Design documentation
