# LED Indicator Boards

## Purpose
Status indication board.
LED indicator boards for embedded devices.

## General features
- Supply voltage: 5V
- Round type LED lamps with a diameter of 5.0 mm
- The connection method to the main board is contacts for soldering wires

- Electrical schematic design
- PCB layout
- Preparation of manufacturing files
- 3D PCB model creation
